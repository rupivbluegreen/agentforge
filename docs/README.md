# docs/

Project documentation: concepts, how-to guides, the compliance guide, and the security
model.

- `adr/` — Architecture Decision Records. One ADR per significant architectural
  decision, numbered (`0001-...`, `0002-...`). Start from the build plan's settled
  decisions (§12) and record new ones as they are made.

A full docs site (concepts, how-to, compliance guide, security model) is built out in
Phase 7.
